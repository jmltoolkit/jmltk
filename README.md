# KeY-JavaParser

![Logo](doc/logo.png)

[![Maven Central](https://img.shields.io/maven-central/v/io.github.jmltoolkit/jmlparser-core.svg)](http://search.maven.org/#search%7Cgav%7C1%7Cg%3A%22io.github.jmltoolkit)

[![Coverage Status](https://codecov.io/gh/javaparser/javaparser/branch/master/graphs/badge.svg?branch=master)](https://app.codecov.io/gh/javaparser/javaparser?branch=master)

This project contains a set of libraries implementing parser and tools for the Java Modeling Language with support for
Java 25.

[Documentation is available](https://jmltoolkit.github.io/)

## Setup

**Gradle**:
```kotlin
implementation("com.github.javaparser:javaparser-symbol-solver-core:3.28.1-J8.0-K13.5-SNAPSHOT")
```

**Maven**:
```xml

<dependency>
    <groupId>io.github.jmltoolkit</groupId>
    <artifactId>jmlparser-core</artifactId>
    <version>3.28.1-J8.0-K13.5-SNAPSHOT</version>
</dependency>
```
